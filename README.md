# ShieldVPN

Secure VPN application for Android.

## About

ShieldVPN provides fast, secure, and reliable VPN connections. Built with Flutter for a smooth native experience.

## Features

- Fast and secure VPN connection
- Multiple protocol support
- Dark minimalist design with neon accents
- Easy server selection
- Real-time connection status

## Tech Stack

- Flutter / Dart
- sing-box core
- Material Design 3

## Colors

- Background: `#020617`
- Primary: `#2563EB`
- Accent: `#22C55E`
- Card: `#0B1120`

## Build

```bash
flutter pub get
flutter build apk
```
